# Steel Structure Design

## 📌 Overview

This project involves the analysis and design of a steel structure under gravity and lateral loads.

## ⚙️ Software

* ETABS
* SAFE
* AutoCAD

## 📊 Description

The structure was modeled and analyzed using ETABS. The project includes load application, structural analysis, and design of steel members based on standard engineering principles.

## 📁 Contents

* Structural model (ETABS)
* Foundation (SAFE)
* Architectural Drawings (AutoCAD)
* Design calculations
* Project report (Persian PDF)
