# Road Geometric Design (Collector–Distributor Road)

## 📌 Overview

This project involves the complete geometric design of a collector–distributor road with a total length of approximately 12 km.

## ⚙️ Software

* AutoCAD
* Civil 3D

## 📊 Description

The project includes the full geometric design of a roadway, covering both horizontal and vertical alignments as well as detailed engineering drawings. Key design components were developed based on standard road design principles.

## 🛣️ Design Components

* Longitudinal profile
* Cross-sections
* Brockenbrough (mass haul) diagram
* Horizontal alignment
* Road signage plan
* Road marking plan

## 📁 Contents

* CAD drawings (plans, profiles, sections)
* Design calculations
* Project report (Persian PDF)

## 📎 Notes

The report is written in Persian; however, design drawings and technical outputs clearly demonstrate the engineering process and results.
