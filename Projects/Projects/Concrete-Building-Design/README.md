# Concrete Building Design

## 📌 Overview

This project involves the analysis and design of a reinforced concrete building under gravity and seismic loads.

## ⚙️ Software

* ETABS
* SAFE
* AutoCAD

## 📊 Description

The structure was modeled and analyzed using ETABS, and foundation design was performed using SAFE. The project includes load analysis, structural modeling, and design of key structural elements.

## 📁 Contents

* Structural model (ETABS)
* Foundation design (SAFE)
